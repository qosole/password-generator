# Password Generator

## Description 

A website with a simple password generator. The website will prompt you for a set of criteria that you want to use for your password.
You can set the length of the password and how many types of characters you want to use.
Types of characters you can select from: lowercase, uppercase, numbers, and special characters.

https://qosole.github.io/password-generator/

2. Follow the on-screen prompts
3. Your password will appear in the box for you to use as you see fit
